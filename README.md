# e-Tendering System
The e-Tendering System is a full-stack  platform designed to streamline the interaction between city administrations, private companies, and citizens. The application provides a transparent environment for managing public tenders, submitting competitive bids, and facilitating civic engagement through feedback loops.

## Tech Stack
- **Frontend:** React
- **Backend:** Node.js with Express
- **Database:** SQL-backed relational architecture
- **DevOps:** Integrated Azure DevOps pipelines for automated CI/CD
- **Architecture:** Multi-role system design supporting City Administrators, Companies, and Citizens

## Key Features
- **Tender Management:** City Administrators can create, update, and monitor public tenders with full control over categories and documentation.
- **Corporate Bidding Portal:** Secure interface for companies to browse open tenders, view detailed requirements, and submit competitive bids.
- **Citizen Engagement:** A specialized dashboard for citizens to view open and closed projects, providing a feedback loop to improve urban transparency.
- **Advanced Validation:** Custom utility layers ensure data integrity across all bidding and registration workflows.
- **Relational Data Modeling:** Comprehensive database design documented via UML and ER diagrams to ensure system scalability.
